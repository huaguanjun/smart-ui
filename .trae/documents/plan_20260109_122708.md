# 清理计划：smart-form-demo 项目

## 目标
清理 smart-form-demo 项目，只保留文档相关逻辑和必要依赖，删除 Vue 应用相关的冗余代码和配置。

## 清理内容

### 1. 删除冗余目录（Vue 应用相关）
- **src/** - Vue 应用源码目录（包含 App.vue、main.ts、components/ 等）
- **dist/** - Vue 应用构建输出目录
- **public/** - Vue 应用静态资源目录
- **.vscode/** - VSCode 配置目录
- **docs/.vuepress/.cache/** - VuePress 缓存目录
- **docs/.vuepress/.temp/** - VuePress 临时文件目录

### 2. 删除冗余文件（Vue 应用相关）
- **index.html** - Vue 应用入口文件
- **vite.config.ts** - Vite 构建配置文件
- **tsconfig.json** - TypeScript 全局配置
- **tsconfig.app.json** - TypeScript 应用配置
- **tsconfig.node.json** - TypeScript Node 配置
- **docs/.vuepress/config.ts.bak** - 配置文件备份

### 3. 修改 package.json
- **保留**所有 VuePress 相关依赖
- **保留**UI 库依赖：`ant-design-vue`、`element-plus`、`smart-form`（smart-ui）
- **保留**文档相关脚本：`docs:dev`、`docs:build`
- **删除**Vue 应用相关脚本：`dev`、`build`、`preview`
- **删除**Vue 应用相关依赖：`@vitejs/plugin-vue`、`vue-tsc`
- **保留**必要的构建依赖：`typescript`、`sass-embedded`
- **删除**未使用的主题：`vuepress-theme-vdoing`
- **更新**描述信息，明确项目用途

### 4. 保留内容
- **docs/** - 完整的文档目录（包含 .vuepress/、api/、guide/ 等）
- **package.json** - 精简后的配置
- **.gitignore** - Git 忽略配置

## 执行步骤

1. 删除指定的冗余目录和文件
2. 修改 package.json，精简脚本和依赖
3. 验证清理后的项目可以正常构建文档

## 预期效果

- 项目结构更清晰，只包含文档相关代码和必要依赖
- 减少项目大小
- 避免混淆，明确项目用途
- 可以正常使用 `npm run docs:dev` 和 `npm run docs:build` 命令
- 文档演示功能正常，支持 element-plus 和 ant-design-vue 适配器切换

## 具体操作清单

### 删除目录
```
rm -rf src/ dist/ public/ .vscode/ docs/.vuepress/.cache/ docs/.vuepress/.temp/
```

### 删除文件
```
rm -f index.html vite.config.ts tsconfig.json tsconfig.app.json tsconfig.node.json docs/.vuepress/config.ts.bak
```

### 修改 package.json
- 保留脚本：`docs:dev`、`docs:build`
- 保留依赖：VuePress 相关、UI 库、必要构建依赖
- 删除依赖：Vue 应用相关依赖、未使用的主题
- 更新描述信息

## 验证方法

执行 `npm run docs:dev` 命令，确认文档服务器可以正常启动，所有演示功能正常显示。
