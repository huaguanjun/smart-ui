# 恢复 package.json 到原始状态

## 问题分析
1. **当前状态**：package.json 中的入口字段指向了源码文件（src/index.ts）
2. **目标状态**：需要将入口字段恢复为指向构建后的文件
3. **影响范围**：main、module、types 和 exports 字段

## 解决方案
将 package.json 中的入口字段恢复为构建后的文件路径，确保项目能正常构建和使用。

## 实施步骤
1. **恢复 main 字段**：从 src/index.ts 改为 dist/smart-ui.umd.cjs
2. **恢复 module 字段**：从 src/index.ts 改为 dist/smart-ui.js
3. **恢复 types 字段**：从 src/index.ts 改为 dist/index.d.ts
4. **恢复 exports 字段**：将所有导出路径从源码文件改为构建后的文件

## 预期效果
- package.json 恢复到原始状态
- 项目能正常构建和使用
- 保持与原有构建流程的兼容性
