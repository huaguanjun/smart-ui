快速开始

安装

npm install smart-form

基本使用

<template>
  <SmartForm 
    :model="formData" 
    :fields="formFields"
    :adapter="adapter"
  />
  <SmartTable 
    :data="tableData" 
    :columns="tableColumns"
    :adapter="adapter"
  />
</template>

<script>
import { SmartForm, SmartTable } from 'smart-form'

export default {
  components: {
    SmartForm,
    SmartTable
  },
  data() {
    return {
      adapter: 'element',
      // 表单配置
      formData: {},
      formFields: [],
      // 表格配置
      tableData: [],
      tableColumns: []
    }
  }
}
</script>

适配器支持

Smart UI 支持多种 UI 框架适配器：

  • Element Plus
  • Ant Design Vue

您可以根据项目需求选择合适的适配器，或参考 Adapter API 自定义适配器。
